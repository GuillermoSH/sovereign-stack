# Homelab Status Page / Pagina de Estado Homelab

## ES

Sitio web para mostrar estado general de mi homelab: hardware, salud basica, disponibilidad y otros datos utiles.

Objetivo: tener punto unico, simple y visual para revisar infraestructura personal.

### Estado del proyecto

- Proyecto activo y en evolucion.
- Se ira actualizando de forma continua.
- Se iran anadiendo nuevas funciones en forma de versiones.

### Roadmap de versiones (idea)

- `v1`: vista base con estado general, imagenes y estructura inicial.
- `v2`: mas secciones de estado (servicios/nodos/metricas clave).
- `v3`: mejoras visuales, contenido dinamico y mas detalle tecnico.

### Tecnologias usadas

- React
- TypeScript
- Vite
- CSS
- ESLint

### Ejecucion local

```bash
npm install
npm run dev
```
